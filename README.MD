# React-express-project skeleton

## About

기본적인 react-express 연동 개발 환경
- babel 을 통한 ES6 변환
- webpack-dev-server 사용
- react-hot-loader 를 통한 Hot Module Reload 사용
- React CodeLab 에서 배포한 프로젝트를 커스터마이징 한 것입니다.
